FlyAway Dedicated Server Shell
List of Commands
Type Each Command out by name, followed by arguments, separated by spaces.
-------------------------------------------------------------------------
exit				            | Exits the Dedicated Server Process
help				            | Shows you this help page
validate {id num}	            | Validates a Certain ID number
exec {sql}			            | [DEBUG] Execute SQL piped to a connection
setuserallow {id num} {0 || 1}	| Set's a user's ability to validate successfully
